Visualizations - normalized temperature stripes¶

Script (make_all_stripes.sh) to regenerate all the stripes plots.

#!/usr/bin/bash

# Make all the stripes plots
# Requires pre-made normalized tensors.
SCRIPT_DIR=$( cd -- "$( dirname -- "${BASH_SOURCE[0]}" )" &> /dev/null && pwd )
cd $SCRIPT_DIR

(cd CRU && ./make_all.sh)

(cd ERA5 && ./make_all.sh)

(cd TWCR && ./make_all.sh)

(cd GPCC/in_situ && ./make_all.sh)

(cd HadCRUT && ./make_all.sh)

(cd HadISST && ./make_all.sh)

(cd GPCP && ./make_all.sh)

The stripes creation scripts are organized by source:

  • ERA5
  • 20CRv3
  • HadISST
  • HadCRUT
